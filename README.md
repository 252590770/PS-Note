## 位图与矢量图

 ### 位图

   * 组成：像素点组成，没有任何保护措施所以变换大小后会失真
   * 优点：颜色非常丰富
   * 缺点：变换大小会失真

 ### 矢量图

   * 组成：路径 + 像素组成 （路径在保护它的像素）
   * 优点：变换大小不失真
   * 缺点：颜色不丰富
